# ⛏️ Simulador Metalúrgico

Aplicación web desarrollada para simular procesos metalúrgicos en minería metálica y no metálica.

## 🚀 Funcionalidades

- 🔹 Cálculo de recuperación metalúrgica
- 🔹 Balance de masa (alimentación, concentrado y relave)
- 🔹 Simulación para:
  - Minería metálica (ej: cobre)
  - Minería no metálica (caliza, sal)
- 🔹 Conversión de unidades (% ↔ ppm)
- 🔹 Selección de elementos:
  - Cobre (Cu)
  - Caliza (Ca)
  - Sal (Na)
- 🔹 Procesos industriales:
  - Lixiviación (hidrometalurgia)
  - Fundición (pirometalurgia)
  - Calcinación
  - Evaporación solar
- 🔹 Visualización de datos mediante gráficos

---

## 🧠 Enfoque del proyecto

Este simulador fue desarrollado integrando conocimientos de ingeniería en metalurgia con desarrollo web, permitiendo representar procesos reales de la industria minera.

Se basa en experiencias relacionadas a operaciones como:

- Procesos de cobre en minería metálica
- Producción de caliza (industria cementera)
- Procesamiento de sal industrial
